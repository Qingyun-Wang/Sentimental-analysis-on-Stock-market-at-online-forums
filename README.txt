This is a webapp that I developed completely from idea generation, data sourcing/collection, model building, deployemnt. It provide the sentimental analysis for the Top
100 largest company in the stock market. It serves users who want to grasp the current mood on these companies in the online forum, specificly for current version on Reddit.
The user can choose the company that their are interested click the botton and get the graph show the movement of the sentiment. 

Instuction:

1) Download this repo.
2) install and activate a virtual environment using the requirement.txt, for linux/mac: 
    python3 -m venv your_venv_name
    source your_venv_name/bin/activate
    pip install -r requirement.txt
3) Run the webapp!
    streamlit run app.py
